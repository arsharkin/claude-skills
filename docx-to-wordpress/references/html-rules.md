# HTML Rules — правила разметки для WordPress классического редактора

## Разрешённые теги

| Тег | Когда использовать |
|-----|--------------------|
| `<p>` | Каждый абзац текста |
| `<h2>` | Подзаголовки раздела (из `##` в markdown) |
| `<ul>` + `<li>` | Маркированный список (из `- ` в markdown) |
| `<ol>` + `<li>` | Нумерованный список (из `1. ` в markdown) |
| `<strong>` | Выделение важного слова внутри текста |

## Запрещено

- `<!-- wp:paragraph -->` и любые Gutenberg block comments
- `<h1>` — заголовок статьи вводится отдельно в поле «Заголовок» WordPress
- `<br>`, `<div>`, `<span>` — классический редактор не нуждается в них
- Пустые строки между тегами

## Пример корректного HTML

```html
<p>Первый абзац текста без отступов.</p>
<ul>
<li>Первый пункт списка;</li>
<li>второй пункт списка.</li>
</ul>
<p>Следующий абзац сразу после списка.</p>
<h2>Подзаголовок раздела</h2>
<p>Текст после подзаголовка.</p>
```

## Нумерованные списки с абзацами внутри пункта

Если пункт нумерованного списка содержит развёрнутое описание (несколько предложений):
```html
<ol>
<li><p><strong>Название пункта.</strong> Описание пункта идёт здесь.</p></li>
<li><p><strong>Второй пункт.</strong> Его описание.</p></li>
</ol>
```
