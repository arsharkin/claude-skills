# Template: чистая версия (для вставки в WordPress)

Структура файла `[название].html`:

```
<!--
  СТАТЬЯ: [Заголовок статьи]

  EXCERPT (поле "Цитата" в WordPress):
  [текст excerpt ~200–250 символов]

  META DESCRIPTION (поле AIOSEO под редактором):
  [текст meta description ~150–160 символов]
-->
[HTML-контент без пустых строк]
```

## Пример

```html
<!--
  СТАТЬЯ: Почему может выбивать автоматический выключатель?

  EXCERPT (поле "Цитата" в WordPress):
  Почему выбивает автоматический выключатель? Разбираем основные причины: перегрузка сети,
  короткое замыкание, неисправность приборов и поломка самого автомата.

  META DESCRIPTION (поле AIOSEO под редактором):
  Почему выбивает автоматический выключатель или дифавтомат? Узнайте об основных причинах
  срабатывания и когда стоит проверить электросеть.
-->
<p>Текст первого абзаца...</p>
<ul>
<li>Первый пункт;</li>
<li>второй пункт.</li>
</ul>
```
