# Template: версия на согласование

Структура файла `[название] — на согласование.html`:

Идентична чистой версии, но каждое исправление выделено тегом `<strong>`.

## Правила выделения

- Добавленное слово или знак препинания: оборачивается в `<strong>...</strong>`
- Исправленное слово: исправленный вариант оборачивается в `<strong>...</strong>`
- Удалённое слово: не применимо (текст слово в слово, удалять нельзя)

## Примеры

Добавлен предлог:
```html
<p>...сказывается не только на электробезопасности, но и <strong>на</strong> помехоустойчивости...</p>
```

Добавлена запятая:
```html
<p>...пайкой<strong>,</strong> например.</p>
```

Исправлено слово:
```html
<p>Итак, основной <strong>функцией</strong> автоматического выключателя...</p>
```

Добавлена запятая вокруг вводного оборота:
```html
<p>Сверхтоки КЗ, возникающие<strong>,</strong> к примеру<strong>,</strong> в случае...</p>
```
